Classes vs id.

What is a class and what is an id?

id and class attributes are used to style elements, they are known as id and class selectors.

Class allows you to use the same css style on multiple elements, where as an id attribute is unique and not reusable. It should only be applied to one element.

It is important to note that id attributes have more importance than a class attribute so if you are applying both a class and id attribute to the same element the styles of the id attribute will always override the class attribute.
