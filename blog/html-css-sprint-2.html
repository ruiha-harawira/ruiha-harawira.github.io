Classes vs id

What is a class and what is an id?

id and class attributes are used to style elements, they are known as id and class selectors

Class allows you to use the same style on multiple elements, where as an id attribute is unique and not reusable it should only be applied to one element. It is important to note that id attributes have more importance than a class attribute so if you are applying both a class and id attribute to the same element the styles of the id will always override the class attribute.

Examples of using id and class attributes into css and HTML:

id elements will always have a # in front of it when applying it in css

    
      "#example-of-id {
           background-color: yellow;
         }"

if you want to put an id element into html it will look like this

  
<h2 id="example-of-id"></h2>

Both print yellow text

Applying a class attribute into CSS looks like this

  
<style> .blue-text { color: blue; }</style>

And finally what a class attribute would look like in HTML

  
<h2 class= "blue-text"></h2>

Both would print blue text
