Javascript and the DOM

HTML is the skeleton of your body, CSS is the skin, Javascript is the brain making everything move and function.

Control flow and loops:

Control flow is the order in which instructions in your programme are executed. Loops allows you to repeat an action a number of times without having to type them in again and again.

In a real life scenario let's say for example a baker that is icing some cookies the action would be

  1. Squeeze a dot of icing onto a cookie
  2. If there are no more cookies, takes a break
  3. Goes back to step 1
And that is repeated until all the cookies are finished being iced. Loops in programming are similar you execute the same instructions until the job is done.

The DOM

The special glue that ties HTML, CSS and java script together is the document object model or the 'DOM' Everything that makes up your DOM is known as nodes These nodes can be anything from:

  • elements
  • Attributes
  • Text content
  • Documents
And many other things that we don't think about

So how do you access html to manipulate it? Depending on what you are wanting to do an easy way to make things easier to grab is by giving them unique ID's, or putting the things you want to manipulate in the same class so that they are easier to access.

We can use a few things like

  • getElementsByTagName(p, div)
  • getElementsByClassName
  • getElementsbyId()
  • getElementsbyName
  • getElementsByTagNameNS

To start off say you are wanting to grab an 

<h1>
element you would use document.querySelector('h1'). If you wanted to grab all of your paragraphs you can use another method like this document.querySelectorAll('p') which will return an array of matching 
<p>
elements otherwise querySelector('p') will return the first paragraph element on your HTML

Another useful tip is changing text in HTML we can use the innerHTML method along with many other useful methods to change and manipulate your HTML. The DOM is not a part of java script language, it is a form of web API. It represents the page so that we can change the structure, style, and content on the page.

Objects vs arrays:

Objects is a container used to display data types that store a collection of data. Arrays are a special type of variable used to store a list of values. We use arrays when we want to create a list of things in a single variable say we want to store all of our favourite foods the array would look something like 

favouriteFoods = [ 'Noodles', 'burgers', 'fruit', 'sushi'];

Here we are listing an array of our favourite foods in a single variable. Now lets make an object for myself it would look something like this 


    var human = { 
      name: 'Ruiha',
      lastName: 'Harawira'
      Age: 25,
    }

Here I have stored a collection of data about myself inside an object. Another example of declaring objects could look something like this we will refer to our javascript cafe tutorial 

    
      let products = {
          whiteCoffee: {
                stock: 4,
                price: 4,
                },
          blackCoffee: {
                stock: 7,
                price: 3.5,
                },

Here we are declaring the object products which include whiteCoffee, blackCoffee and the price and stock of each of those items.
